# indicacoesdoclubedolivro
Este guia reúne as indicações de leitura feitas pelos próprios membros da comunidade. Foi montado para facilitar a pesquisa e a escolha de leituras, caso alguém queira conhecer melhor um livro antes de começar. Todas as indicações foram sugeridas organicamente pelo grupo. Quer indicar um livro? Use o formulário no final da página!
